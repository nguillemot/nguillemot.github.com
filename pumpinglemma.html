Pumping Lemma for Regular Languages

Let L be a regular language. There is an integer p ≤ 1 such that any string w ∈ L with |w| ≥ p can be rewritten as w = xyz such that y ≠ ε, |xy| ≤ p, and xyiz ∈ L for each i ≥ 0.

The five parts of this theorem can be restated in the following way:

(1) for each regular language L,
(2) there exists a p ≥ 1, such that
(3) for each string w ∈ L with |w| ≥ p,
(4) there exist strings x, y, z with w = xyz, y ≠ ε, and |xy| ≤ p, such that
(5) for each i ≥ 0, xyiz ∈ L.

Example: L = { aibi : i ≥ 0 }

  • By (1), we begin by assuming that L is regular.
  • (2) says that if we can find a single p that satisfies (3)-(5), then L is indeed regular.
  • For every p ≥ 1, we can build a string w ∈ L with |w| ≥ p in the form of w = apbp. In these cases, |w| = 2p, which always satisfies |w| ≥ p. Therefore, (4)-(5) should always be possible to satisfy for w.
  • Since |xy| ≤ p and the first p symbols of w are all a, it must be that y = ak where 0 < k ≤ p. It follows that x = ap - k and z = bp. There must then exist a k for which (5) is satisfied.
  • No matter what choice of k is made, setting i = 0 generates the string xz = ap - kbp. Since p - k ≠ p, this string is not in L. Therefore, we cannot satisfy (5), which means we have found a contradiction.
This shows that no matter what p is chosen, we can always find a string which will satisfy the requirements of (3) but fail to satisfy the requirements of (4)-(5). Therefore, L is not a regular language.
